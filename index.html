BatGen

A simple yet powerful tool for generating SQL, MyBatis files, and the supporting DAO/BO objects all at once.

Documentation »

Get coding faster, simplify the basics

  • Avoid creating dozens of SQL files.
  • Have the Data Access Objects and the Business Objects generated for you!
  • Easily refactor database changes across DAOs and BOs.
  • Batgen generates the CRUD commands, builds your domain objects and creates the mappers for mybatis so you can focus on the important things.

Try BatGen!

Just format your variables into a text file to automate these tedious tasks.
